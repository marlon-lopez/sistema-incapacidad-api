# Sistema de Incapacidad API

> Backend API for Sistema-Incapacida application

## Stack

Node Js, Express Js, mongoDB, mongoose, JWT (JSON Web Tokens)

## Install dependencies

```
npm install

```

## Run App

```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
